1. Нарушение цели безопасности №1
Только авторизованные пользователи могут взаимодействовать со спутником

Основная ценность: предотвращение раскрытия коммерческой тайны

Негативный сценарий: злоумышленник получает доступ к системе и запрашивает снимки

Код для plantuml:
@startuml
title НС1 — Неавторизованный пользователь получает доступ к спутнику

actor "Злоумышленник" as Att
participant "Система спутника" as S
participant "Модуль аутентификации" as Auth
participant "Камера" as Cam

Att -> S: Запросить снимок зоны X
S -> Auth: Проверка аутентификации
Auth --> S: Отказ — пользователь неавторизован

alt Уязвимость в проверке доступа
    note right: Оборудование работает штатно (предположение №1)\nОшибка находится в логике авторизации.
    S -> Cam: Сделать снимок зоны X
    Cam --> S: Снимок получен
    S -> Att: Передача снимка (утечка коммерческой тайны)
end

@enduml

2. Нарушение цели безопасности №2
Пользователи могут выполнять только разрешённые операции

Основная ценность: предотвращение ломки спутника

Негативный сценарий: пользователь базового уровня выполняет команду изменения орбиты

Код для plantuml:

@startuml
title НС2 — Пользователь выполняет запретные операции (коррекция орбиты)

actor "Клиент (базовый доступ)" as U
participant "Система спутника" as S
participant "Модуль прав доступа" as Rights
participant "Орбитальный модуль" as OM

U -> S: Команда изменить орбиту
S -> Rights: Проверка прав пользователя
Rights --> S: Прав недостаточно

alt Уязвимость в проверке прав
    note right: Предположение №1 выполняется — оборудование исправно.\nОшибка — только в логике авторизации/контроля.
    S -> OM: Выполнить манёвр
    OM --> S: Орбита изменена в опасную область
    note right: Нарушение безопасности спутника.
end

@enduml

3. Нарушение цели безопасности №3
Спутник не делает фото запрещённых районов

Основная ценность: защита коммерческой тайны

Негативный сценарий: пользователю удаётся сфотографировать запрещённый район

Код для plantuml:
@startuml
title НС3 — Несанкционированная съёмка запрещённой зоны

actor "Клиент" as C
participant "Система спутника" as S
participant "Модуль ограничений зон" as ZM
participant "Камера" as Cam

C -> S: Запросить снимок зоны X
S -> ZM: Проверка: зона запрещена?
ZM --> S: Зона запрещена

alt Некорректная реализация проверки зоны
    S -> Cam: Сделать снимок зоны X
    Cam --> S: Снимок получен
    S -> C: Передача запрещённого снимка
    note right: Утечка коммерческой тайны — ключевая негативная цель.
end

@enduml

4.Нарушение цели безопасности №4
Орбита спутника всегда остаётся в рамках ограничений

Основная ценность: сохранность спутника и отсутствие вреда людям

Негативный сценарий: пользователь с особым доступом изменяет орбиту так, что она выходит за пределы ограничений

Код для plantuml:
@startuml
title НС4 — Спутник выводится за пределы допустимой орбиты

actor "Клиент (особый доступ)" as U
participant "Система спутника" as S
participant "Модуль орбитальных ограничений" as Limits
participant "Орбитальный модуль" as OM

U -> S: Команда изменить орбиту (опасные параметры)
S -> Limits: Соответствует ограничениям?
Limits --> S: Параметры не соответствуют

alt Уязвимость — проверка ограничений не сработала
    note right: Оборудование исправно (предположение №1)
    S -> OM: Выполнить манёвр
    OM --> S: Орбита выходит за пределы установленной зоны
    note right: Риск разрушения спутника и вторичных инцидентов.
end

@enduml
